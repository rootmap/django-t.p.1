Instruction for DJango :

  • Step 1 : Install Python from internet
  • Step 2 : Install PyCharm Communinity Edition from internet
  • Step 3 : Create New Project in pycharm
  • Step 4 : Open Terminal in pycharm and type
  • Step 4.1 : pip install django
  • Step 5 : check Django Version if successfully Installed
  • Step 5.1 : python -m django --version
  • Step 6 : Start new project for django Type Command in terminal
  • Step 6.1 : django-admin startproject DEMOPROJECT
  • Step 7 : Now Run Python webserver to get django webpage | Type Below Command
  • Step 7.1 : python manage.py runserver
  • Step 8 : Now create new app and can create n number of app
  • Step 8.1 : Type command in terminal
  • Step 8.2 : python manage.py startapp DEMOAPP
  • Step 9 : python manage.py runserver
